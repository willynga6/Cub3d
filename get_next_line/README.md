# GET_NEXT_LINE

**Get_next_line** is a project from the first milestone of the 42 school.  
The objective is to implement a function that reads a file **line by line**, regardless of the file size.

---

## 📌 Project Goal

The function must return a line ending with a newline character (`\n`) each time it is called,  
or `NULL` when there is nothing left to read.
